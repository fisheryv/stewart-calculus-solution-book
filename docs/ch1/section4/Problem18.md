<h4 class="alert-heading">Problem 18</h4>

Find the domain of each function.

1. $\displaystyle g(t) = \sqrt{10^t-100}$

2. $\displaystyle g(t) = \sin (e^t-1)$

</div>

<div class="alert alert-success" role="alert">
<h4 class="alert-heading">Solution</h4>

1. Since the radicand must be greater than or equal to $0$, therefore

    $$
    10^t-100 \ge 0 \implies 10^t \ge 100 \implies t \ge 2
    $$

    The domain of function $g(t)$ is $[2, \infty)$.

2. Since the domain of $\sin()$ and exponential function are both $\mathbb{R}$, therefore the domain of function $g(t)$ is $\mathbb{R}$.
