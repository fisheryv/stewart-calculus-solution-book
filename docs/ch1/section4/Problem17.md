<h4 class="alert-heading">Problem 17</h4>

Find the domain of each function.

1. $\displaystyle f(x) = \frac{1-e^{x^2}}{1-e^{1-x^2}}$

2. $\displaystyle f(x) = \frac{1+x}{e^{\cos x}}$

</div>

<div class="alert alert-success" role="alert">
<h4 class="alert-heading">Solution</h4>

1. The domain of exponential function is $\mathbb{R}$, we only need to ensure that the denominator of the function $f(x)$ is not zero, that is

    $$
    1-e^{1-x^2} \ne 0 \implies e^{1-x^2} \ne 1 \implies 1-x^2 \ne 0 \implies x \ne \pm 1
    $$

    Therefore, the domain of function $f(x)$ is $\{x \mid x \in \mathbb{R} \land x \ne \pm 1\}$

2. The domain of exponential function is $\mathbb{R}$, we only need to ensure that the denominator of the function $f(x)$ is not zero. And we know that the range of exponential function is $(0, \infty)$, so the denominator $e^{\cos x}$ is always greater than $0$. 

    Therefore, the domain of function $f(x)$ is $\mathbb{R}$.

</div>
